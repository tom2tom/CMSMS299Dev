• 2014 - Initial release
  • May 2022 Version 2.0
    • Performance tuning
    • Most navigation data are recorded in the system cache and refreshed asynchronously
    • Former 'extended' navigation parameters are always available
    • Processing arrays of node-objects in Smarty navigation-templates is deprecated in favour of processing arrays of node-ids.
      Broadly, each occurrence of
      {foreach $whatever as $node}

      in such templates becomes
      {foreach $whatever as $id}{$node = $nodes.$id}
    • Until node-object templates are abandoned, include parameter 'idnodes'=1 in any tag which initiates processing of a navigation-template that uses node-id's instead
