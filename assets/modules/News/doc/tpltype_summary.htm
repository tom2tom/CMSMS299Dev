Summary-Template Help

The News module's 'default' action exports the following variables:

  • $prevpage (string) - If the current page number is 1, this is a translated string for the 'prevpage' key. Otherwise, it is a link that will result in a summary view displaying the previous page.
  • $prevurl (string) - If the current page number is 1, this is not defined/empty. Otherwise it is an URL that when placed in an <a> tag and clicked on will result in a summary view displaying the previous page.
  • $firstpage (string) - If the current page number is 1, this is a translated string for the 'fistpage' key. Otherwise, it is a link that will result in a summary view displaying the fist page.
  • $firsturl (string) - If the current page number is 1, this is not defined/empty. Otherwise it is a URL that when placed in an <a> tag and clicked on will result in a summary view displaying the first page.
  • $nextpage (string) - If the current page number is the last page given the current criteria, this is a translated string for the 'nextpage' key. Otherwise, it is a link that will result in a summary view displaying the next page.
  • $nexturl (string) - If the current page number is the last page givent the current criteria, this is not defined/empty. Otherwise it is an URL that when placed in an <a> tag and clicked on will result in a summary view displaying the next page.
  • $lastpage (string) - If the current page number is the last page given the current criteria, this is a translated string for the 'nextpage' key. Otherwise, it is a link that will result in a summary view displaying the last page.
  • $lasturl (string) - If the current page number is the last page givent he current criteria, this is not defined/empty. Otherwise it is an URL that when placed in an <a> tag and clicked on will result in a summary view displaying the last page.
  • $pagenumber (int) - The current page number. Starting from 1.
  • $pagecount (int) - The total number of pages to display, given the current criteria
  • $oftext (string) - The translated string for "of"
  • $pagetext (string) - The translated string for "page"
  • $itemcount (int) - The total number of items that will be displayed in this view (count of the $items array)
  • $items (object[]) - An array of objects representing the news articles to be displayed
  • $category_name (string) - If a category id or category name was passed to the call to this module, then this variable will contain the name of the specified category. Otherwise, it will be empty.
  • $cats (array) (deprecated) - An array of category information representing all categories in the News module

News Article Information

Each member of the $items array is a PHP object.

Each such object's members/properties are listed below.

  • author (string) The username of the author... or unknown
  • author_id (int) - The uid of the author. A value of 0 indicates an anonymous user
  • authorname (string) - A concatenation of the users first and last names, if they exist
  • category (string) - The article's category name.
  • content (string) - The HTML content for the article
  • create_date (string) - The date/time when the article was first created
  • detail_url (string) - The URL to the detail view of the article
  • enddate (string) - Optional, the expiry date of the article
  • extra (string) - Optional, extra information
  • id (int) - The numeric id of the article
  • link (string) (deprecated) - The URL to the detail view of the article
  • modified_date (string) - The date/time when the article was last modified
  • morelink (string) (deprecated) - A link (using a translation of the word "more" as the text), to the detail view of the article
  • moreurl (string) (deprecated) - A URL to the detail view of the article.
  • startdate (string) - Optional, the publication date of the article
  • summary (string) - The article summary
  • title (string) - The title of the article
  • titlelink (string) (deprecated) - A link (using the article title as the text) to the detail view of the article

These article properties may be used in a template like {$entry->id} etc.

Category Information

Each member of the $cats array is itself an associative array.

The following are the notable members of such arrays. Other members may also be present for internal organizational purposes.

Use of this information for displaying categories-information is deprecated. Instead use module's 'browsecat' action

  • category_url (string) - A slug which may be included in a pretty-URL for linking to the category
  • count (int) - The number of displayable articles in the category
  • depth (int) - The depth (from the root category) of the current node
  • image_url (string) - The uploads-folder-relative URL of an associated image (might be empty or unset)
  • news_category_id (int) - The numeric id of the category
  • news_category_name (string) - The name of the category
  • parent_id (int) - The numeric id of the category's parent. A value of -1 indicates no parent.
  • prevdepth (int) - The depth (from the root) of the previous node
  • url (string) - URL for showing a summary-view of all articles in the category

This category information may be used in a template like {$cats[0]['news_category_name']} or {$cats.0.news_category_name} etc.
